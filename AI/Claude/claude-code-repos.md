# 10 GitHub Repositories to Level Up Claude Code

Curated list of community repositories that extend Claude Code with agents, skills, hooks, subagents, templates, and best practices.

> **New to Claude Code?** Follow the [Beginner to Ninja step-by-step guide](step_by_step_guide.md) to learn these repos in the right order.

---

## 1. Claude Swarm — affaan-m

**Repo:** https://github.com/affaan-m/claude-swarm

A full performance-focused agent harness built from 10+ months of real-world use. Covers agents, skills, hooks, MCP configs, memory optimization, and more. Multi-agent orchestration with a rich terminal UI for visualizing task decomposition and coordination.

## 2. System Prompts and Models of AI Tools — x1xhlol

**Repo:** https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools

Compare the internal system prompts of Claude Code, Cursor, Devin, Replit, Windsurf, Augment Code, and others. A goldmine for prompt designers — includes full system prompts, internal tools, and AI model details extracted from each tool.

## 3. GStack — garrytan

**Repo:** https://github.com/garrytan/gstack

Garry Tan's personal Claude Code setup — structures AI into organizational roles (CEO, Designer, Engineering Manager, QA, Release Manager, Doc Engineer) using reusable skills and slash commands. 23 opinionated tools ready to drop into your workflow.

## 4. Get-Shit-Done — gsd-build

**Repo:** https://github.com/gsd-build/get-shit-done

Breaks work into structured stages: discussion → planning → execution → verification → shipping. Perfect for larger, complex projects where you need Claude Code to follow a disciplined workflow rather than jumping straight to code.

## 5. Learn-Claude-Code — shareAI-lab

**Repo:** https://github.com/shareAI-lab/learn-claude-code

Build a Claude Code–style agent harness from scratch. Covers agent loops, subagents, context compression, and git worktree isolation. Ideal for understanding how Claude Code works under the hood so you can customize or extend it.

## 6. Awesome-Claude-Code — hesreallyhim

**Repo:** https://github.com/hesreallyhim/awesome-claude-code

The ultimate curated directory of skills, hooks, slash commands, agent frameworks, and plugins in the Claude Code ecosystem. A one-stop hub for discovering community contributions.

## 7. Claude-Code-Templates — davila7

**Repo:** https://github.com/davila7/claude-code-templates

Ready-made configs for agents, hooks, MCP integrations, and project templates. Saves hours on initial setup — pick a template, drop it in, and start coding.

## 8. Claude-Code-Best-Practice — shanraisshan

**Repo:** https://github.com/shanraisshan/claude-code-best-practice

A hands-on playbook for working smarter with Claude Code — covers commands, subagents, hooks, and project instructions. Practical patterns distilled from real production use.

## 9. Awesome-Claude-Code-Subagents — VoltAgent

**Repo:** https://github.com/VoltAgent/awesome-claude-code-subagents

A large library of specialized subagent definitions for real development tasks. See role specialization in action — each subagent is tailored for a specific workflow (code review, testing, deployment, etc.).

## 10. Claude-Code-System-Prompts — Piebald-AI

**Repo:** https://github.com/Piebald-AI/claude-code-system-prompts

Tracks Claude Code's internal prompts, tool descriptions, and changes across versions. Essential for prompt researchers who want to understand what Claude Code tells itself and how it evolves between releases.
